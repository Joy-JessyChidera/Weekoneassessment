Html Formatting Tags

This is a list of HTML formatting tags

Name Tags
Bold text <b>- This is a physical tag used to bold the text written between it.
Important <strong> This is a logical tag which tells the browser that the text is important.
Italic- <i>-This a physical tag which is used to make text italic
Emphasized- <em>-This a logical tag which is used to display content in italic.
Marked text <mark>-This tag is used to highlight text.
Smaller text <small>-This tag is used to decrease the font size by one conventional unit.
Deleted text <del>- This tag is used to display the deleted content.
Inserted text <ins>-This tag displays the content which is added.
Subscript text <sub>-It displays the text slightly below the normal line.
Superscript <sup>-It displays the content slightly above the normal line.
Big <big>-This tag is used to increase the font size by one conventional unit.
